# FlowNote 运维部署手册

> 常用命令速查，涵盖后端部署、前端部署、文件传输、故障排查等。

---

## 一、后端部署（服务器）

### 1.1 进入目录
```bash
cd ~/video-notes/backend
```

### 1.2 激活虚拟环境
```bash
source venv/bin/activate
```

### 1.3 安装/更新依赖
```bash
pip install -r requirements.txt
```

### 1.4 启动后端（开发模式）
```bash
uvicorn app.main:app --host 0.0.0.0 --port 8001
```

### 1.5 后台启动（nohup）
```bash
nohup uvicorn app.main:app --host 0.0.0.0 --port 8001 > app.log 2>&1 &
```

### 1.6 停止后端
```bash
# 查找并杀掉 8001 端口进程
fuser -k 8001/tcp

# 或手动查找再杀
ps -ef | grep 8001
kill -9 <PID>
```

### 1.7 查看后端日志
```bash
tail -f app.log
tail -f app.log | grep ERROR
```

## 三、文件传输（scp）

### 3.1 本地 → 服务器（上传）

**上传单个文件**
```bash
# Windows PowerShell / Git Bash
scp /本地/路径/文件名.txt ubuntu@118.25.22.220:/home/ubuntu/video-notes/backend/

# 示例：上传后端文件
scp backend/app/routers/tasks.py ubuntu@118.25.22.220:/home/ubuntu/video-notes/backend/app/routers/

# 示例：上传前端文件
scp frontend/src/app/dashboard/page.tsx ubuntu@118.25.22.220:/home/ubuntu/video-notes/frontend/src/app/dashboard/
```

**上传整个目录**
```bash
scp -r /本地/路径/目录名 ubuntu@118.25.22.220:/home/ubuntu/video-notes/

# 示例：上传整个 backend 目录
scp -r backend/ ubuntu@118.25.22.220:/home/ubuntu/video-notes/

# 示例：上传整个 frontend/src/app 目录
scp -r frontend/src/app/ ubuntu@118.25.22.220:/home/ubuntu/video-notes/frontend/src/
```

### 3.2 服务器 → 本地（下载）

**下载单个文件**
```bash
scp ubuntu@118.25.22.220:/home/ubuntu/video-notes/backend/videonotes.db ./备份/
```

**下载整个目录**
```bash
scp -r ubuntu@118.25.22.220:/home/ubuntu/video-notes/backend/ ./备份/
```

### 3.3 压缩后上传（推荐大文件/整个项目）

**本地压缩**
```bash
# Windows PowerShell
Compress-Archive -Path frontend/* -DestinationPath frontend.zip

# 或 Git Bash
zip -r frontend.zip frontend/ -x "frontend/node_modules/*" -x "frontend/.next/*"
```

**上传压缩包**
```bash
scp frontend.zip ubuntu@118.25.22.220:/home/ubuntu/video-notes/
```

**服务器解压**
```bash
cd ~/video-notes
unzip frontend.zip
```

## 五、网络与防火墙

### 5.1 查看端口占用
```bash
ss -tlnp | grep 8001
lsof -i :8001
```

### 5.2 测试后端 API
```bash
curl http://118.25.22.220:8001/api/health
curl http://localhost:8001/api/health
```

### 5.3 查看服务器防火墙状态
```bash
sudo ufw status
sudo iptables -L -n | grep 8001
```

### 5.4 腾讯云控制台放行端口
- 轻量服务器：登录腾讯云控制台 → 防火墙 → 添加规则 → TCP 8001/3000
- HAI GPU：腾讯云控制台 → HAI 实例 → 安全组 → 放行 8000

## 七、故障排查

### 7.1 后端启动失败
```bash
# 检查依赖是否安装
pip list | grep fastapi

# 检查端口是否被占
lsof -i :8001

# 检查配置文件
ls -la ~/video-notes/backend/.env
cat ~/video-notes/backend/.env
```

### 7.2 前端请求后端超时
- 确认后端已启动：`curl http://118.25.22.220:8001/api/health`
- 确认服务器防火墙放行 8001 端口
- 确认 `next.config.js` 中的 rewrite 地址正确

### 7.3 字幕解析失败
- B 站 API 偶尔限流，稍等重试
- 确认视频链接格式正确（包含 BV 号）

## 八、快速重启全流程

```bash
# 1. 上传代码（本地执行）
scp -r backend/ frontend/ ubuntu@118.25.22.220:/home/ubuntu/video-notes/

# 2. SSH 登录服务器
ssh ubuntu@118.25.22.220

# 3. 重启后端
cd ~/video-notes/backend
source venv/bin/activate
fuser -k 8001/tcp
nohup uvicorn app.main:app --host 0.0.0.0 --port 8001 > app.log 2>&1 &

# 4. 重启前端（新终端）
cd ~/video-notes/frontend
npm install
fuser -k 3000/tcp 2>/dev/null
nohup npm run dev > frontend.log 2>&1 &

# 5. 验证
curl http://localhost:8001/api/health
```
