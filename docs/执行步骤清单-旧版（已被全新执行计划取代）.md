# 视频笔记工具 — 执行步骤清单

> 目标：从零启动，到正式上线收费，预计 6 周完成  
> 原则：备案与开发并行，不等备案再动手

---

## 阶段一：立刻执行（第 1 周）

### 1.1 腾讯云账号与资质

- [ ] 注册腾讯云账号（https://cloud.tencent.com）
- [ ] 完成企业实名认证
  - 材料：营业执照扫描件 + 法人身份证正反面
  - 审核时间：约 1 个工作日
- [ ] 购买轻量应用服务器
  - 配置：2核4G，大陆节点（推荐上海或广州）
  - 时长：至少购买 1 年（约 ¥80~100/年）
  - 注意：必须是包年包月、有公网 IP，才满足备案条件

### 1.2 域名

- [ ] 在腾讯云购买域名（推荐 `.com`，约 ¥80/年）
- [ ] 完成域名实名认证（企业主体）
- [ ] 确认产品名称后再买，备案后改域名麻烦

### 1.3 提交 ICP 备案

- [ ] 进入腾讯云备案控制台，生成备案授权码
- [ ] 填写企业主体信息（营业执照 + 法人身份证）
- [ ] 提交腾讯云初审（1~2 个工作日）
- [ ] 初审通过后，完成工信部短信核验（24 小时内）
- [ ] 等待管局审核（各省不同，上海约 3 天，北京约 15 天）

### 1.4 微信开放平台（与备案并行）

- [ ] 注册微信开放平台账号（https://open.weixin.qq.com）
- [ ] 提交企业资质认证
  - 材料：营业执照 + 法人身份证 + 对公账户信息
  - 认证费：¥300/年
  - 审核时间：约 3~5 个工作日
- [ ] 认证通过后，创建「网站应用」，用于微信登录

---

## 阶段二：开发期（第 1~4 周，与备案并行）

> 用服务器 IP 地址访问调试，不依赖域名

### 2.1 服务器基础环境

- [ ] SSH 登录服务器，更新系统
- [ ] 安装 Node.js（v20+）和 Python（3.10+）
- [ ] 安装并配置 Nginx
- [ ] 配置 Git，关联 GitHub 代码仓库
- [ ] 安装 PM2（Node 进程管理）或 Supervisor（Python）
- [ ] 安装 Redis（任务队列）

### 2.2 后端核心功能开发

- [ ] 搭建项目基础框架（推荐 FastAPI 或 Node.js + Express）
- [ ] 实现 B 站链接解析模块
  - 优先抓取内嵌 CC 字幕（零成本）
  - 无字幕时提取音频，准备走 ASR
- [ ] 接入腾讯云 ASR（录音文件识别接口）
  - 开通服务，获取 API Key
  - 实现异步任务：提交 → 轮询结果 → 回调通知
- [ ] 接入 LLM API（推荐 DeepSeek，国内访问稳定）
  - 实现 AI 总结 Prompt
  - 实现结构化笔记生成 Prompt
  - 实现 Markdown 格式输出
- [ ] 搭建任务队列（Celery + Redis 或 BullMQ）
  - 视频处理异步化，用户提交后后台处理
  - 处理完成后通知前端（WebSocket 或轮询）
- [ ] 用户系统（邮箱注册 + JWT 鉴权，微信登录备案后补）
- [ ] 使用次数限制逻辑（免费层每月 3 次）

### 2.3 前端开发

- [ ] 搭建前端项目（推荐 Next.js）
- [ ] 首页：产品介绍 + 链接输入框
- [ ] 处理页：进度展示（进度条 + 状态文案）
- [ ] 结果页：
  - 全文转录（带时间戳）
  - AI 总结
  - 结构化笔记
  - Markdown 一键导出按钮
- [ ] 用户中心：历史记录、剩余次数展示
- [ ] 定价页：免费 / 学生版 / 无限版

### 2.4 内测

- [ ] 用 IP 地址邀请 5~10 位真实学生内测
- [ ] 收集反馈，记录高频问题
- [ ] 优化转录准确率和笔记质量

## 阶段三：备案通过后上线（第 4~6 周）

### 3.1 域名与 HTTPS

- [ ] 备案通过后，将域名解析到服务器 IP
- [ ] 在腾讯云申请免费 SSL 证书（DV 证书，即时签发）
- [ ] 配置 Nginx，强制 HTTPS 跳转

### 3.2 接入微信登录

- [ ] 在微信开放平台「网站应用」中，填写已备案域名作为授权回调域
- [ ] 后端实现 OAuth 2.0 授权流程
- [ ] 前端添加「微信登录」按钮
- [ ] 测试完整登录链路

### 3.3 接入支付

- [ ] 申请微信支付商户号（https://pay.weixin.qq.com）
  - 材料：营业执照 + 法人身份证 + 对公账户
  - 审核：1~2 个工作日
- [ ] 申请支付宝商家账号
- [ ] 后端实现订阅扣费逻辑（包月/包年）
- [ ] 前端实现付费弹窗与套餐选择页

### 3.4 合规收尾

- [ ] 网站底部挂 ICP 备案号
- [ ] ICP 备案通过后 30 天内，完成公安联网备案（免费）
- [ ] 添加用户协议和隐私政策页面

### 3.5 正式上线

- [ ] 全流程回归测试（注册 → 提交视频 → 生成笔记 → 付费订阅）
- [ ] 配置服务器监控和告警（腾讯云云监控）
- [ ] 开放公开注册
