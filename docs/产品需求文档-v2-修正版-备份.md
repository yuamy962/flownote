# 视频笔记工具 — 产品需求文档 v2.0（修正版）

> 版本：v2.0  
> 目标用户：在校大学生、考研/考证/技术学习者  
> 核心定位：低成本本地转录 + AI 结构化笔记工具，定价对标飞书妙记降低 70%  
> 文档状态：融合 ChatGPT/Claude 双版本优势，修正商业模式与成本模型

## 一、产品概述

### 1.1 解决的问题

学生观看 B 站课程、网课录播时，手动记笔记效率低、容易遗漏关键知识点。市面现有工具（飞书妙记 ¥28/月、通义听悟 ¥30/月）价格超出学生预算。

### 1.2 核心价值

- **零边际成本转录**：B 站内嵌字幕直接解析（约 70% 视频有字幕）；无字幕视频走本地 Whisper，固定成本可控
- **AI 结构化笔记**：自动生成标题、要点、代码块、补充说明，适配复习场景
- **Markdown 导出**：一键导出 `.md`，兼容 Notion / Obsidian / Typora
- **学生友好定价**：基础功能免费，付费版按**转录时长**计费，不用焦虑"视频个数"

### 1.3 目标用户画像

| 用户类型 | 特征 | 典型场景 |
|----------|------|----------|
| 大学生 | 用 B 站学专业课、考研、四六级 | 1~2 小时课程视频 |
| 考证人群 | CPA、法考、教师资格证 | 网盘录播课 |
| 技术学习者 | 编程、AI、运维教程 | 30~60 分钟技术视频 |
| 内容创作者 | 整理视频素材、提取文案 | 自有视频二次加工 |

## 二、功能需求

### 2.1 输入方式

| 输入方式 | 优先级 | 说明 |
|----------|--------|------|
| B 站视频链接 | P0 | 粘贴 URL，优先解析 CC 字幕；无字幕则提取音频转录 |
| 本地文件上传 | P0 | 支持 MP4、MOV、MP3、WAV，最大 2GB |
| 百度网盘 | P1 | OAuth 授权后选取文件（二期） |
| 阿里网盘 | P1 | OAuth 授权后选取文件（二期） |
| 其他链接（YouTube 等） | P2 | 仅支持字幕解析，不支持下载（合规） |

### 2.2 核心功能

#### 2.2.1 语音转文字（转录）

- **B 站字幕优先**：自动检测内嵌 CC 字幕，有则直接解析（零成本、秒出结果）
- **本地 Whisper 转录**：无字幕视频通过 ffmpeg 提取音频，本地 Faster-Whisper 转录
- **时间戳对齐**：转录结果按段落带时间戳，点击可跳转视频对应位置
- **语言支持**：普通话、粤语、英语（依赖 Whisper 模型能力）
- **质量过滤**：音频清晰度低于阈值（如信噪比过低、纯音乐）自动拒绝，提示用户

#### 2.2.2 AI 总结

- 提取视频核心观点（3~5 条）
- 按章节自动分段摘要
- 关键词高亮标注
- 输出"适合人群"与"学习建议"

#### 2.2.3 自动生成笔记

- 输出结构化笔记：标题 / 核心要点 / 代码示例 / 补充说明
- 笔记格式适配学生复习（大纲式、康奈尔式可选）
- 支持在线编辑与手动调整

#### 2.2.4 Markdown 导出

- 一键导出 `.md` 文件
- 格式兼容 Notion、Obsidian、Typora
- 支持复制全文到剪贴板
- 导出内容包含：转录原文、AI 总结、结构化笔记（可选模块）

#### 2.2.5 视频问答（P1）

- 基于转录内容进行 RAG 问答
- 回答引用原文时间戳
- 付费版专属功能

#### 2.2.6 闪卡生成（P2）

- 自动生成 Anki 格式复习卡片
- 支持 `.apkg` 导出

### 2.3 用户系统

- 邮箱注册 + 密码登录（邮箱需验证）
- 微信一键登录（OAuth 2.0，备案通过后接入）
- JWT Token 鉴权，Access Token 有效期 7 天，Refresh Token 有效期 30 天
- 历史记录保留最近 90 天（非 30 条，避免用户删除后无法找回）
- 支持软删除，删除后 7 天内可恢复

## 三、非功能需求

### 3.1 性能

| 场景 | 指标 |
|------|------|
| 视频提交后首屏反馈 | ≤ 10 秒（返回任务ID与排队位置） |
| B站视频有字幕 | 处理完成 ≤ 30 秒 |
| 30 分钟视频（本地Whisper CPU） | 处理完成 ≤ 15 分钟 |
| 1 小时视频（本地Whisper CPU） | 处理完成 ≤ 30 分钟 |
| 页面首屏加载 | ≤ 2 秒 |
| API 响应时间（非转录类） | ≤ 500ms |

> 注：本地 CPU 转录速度受限于服务器算力，高峰期可能出现排队。专业版/无限版享有优先队列。

### 3.2 可用性

- 服务可用性目标：99.5%（每月允许宕机 ≤ 3.6 小时）
- 转录任务失败时，自动重试 1 次；仍失败则标记为失败并通知用户
- 每日凌晨 3:00 自动备份数据库到 COS

### 3.3 安全

- 用户密码 bcrypt 加密存储
- 视频/音频文件处理完成后 24 小时内删除，不长期存储原文件
- 转录结果与笔记保留 90 天，超期自动清理（用户导出后不受此限制）
- HTTPS 全站加密
- API 限流：
  - 普通接口：同一 IP 60 次/分钟
  - 转录提交：同一用户 5 次/小时（免费版）、10 次/小时（付费版）
  - 注册接口：同一 IP 5 次/小时，需图形验证码
- 上传文件类型白名单校验（MP4/MOV/MP3/WAV），大小限制 2GB

### 3.4 反滥用机制

| 策略 | 说明 |
|------|------|
| 设备指纹 | 同一设备注册多账号，免费时长共享上限 |
| 行为检测 | 短时间内批量提交相似视频（如相同视频反复上传），触发人工审核 |
| 公平使用 | 无限版单月超过 50 小时，次月自动降级为专业版（可申诉） |
| 黑名单 | 疑似脚本/爬虫的账号，限制转录功能 |

### 3.5 合规

- ICP 备案（企业主体）
- 公安联网备案（ICP 通过后 30 天内完成）
- 隐私政策 + 用户协议（明确：仅限个人学习用途，禁止批量爬取、禁止内容再分发）
- 不存储视频原文件，处理完即删
- 不向第三方出售用户数据

```

    │

Nginx（反向代理 + HTTPS + 静态资源）
```

### 4.4 语音转录方案（核心修正点）

| 方案 | 使用场景 | 成本 | 速度 |
|------|----------|------|------|
| B 站内嵌字幕解析 | B 站视频有 CC 字幕 | ¥0 | 秒级 |
| 本地 Faster-Whisper (CPU) | 无字幕视频、本地上传 | 服务器固定成本，边际≈¥0.07/小时 | 实时因子 0.5~1.0x |
| 本地 Faster-Whisper (GPU，二期) | 用户量大后升级 | GPU 服务器固定成本 | 实时因子 10~20x |

**B 站处理流程：**
1. 请求 B 站 API 获取视频 CID
2. 检测 CC 字幕（`/x/player/v2` 接口）
3. **有字幕**：下载 JSON 字幕，解析为带时间戳文本，直接送入 LLM
4. **无字幕**：ffmpeg 提取音频 → 上传 COS 临时存储 → Celery 队列调度本地 Whisper 转录 → 结果送入 LLM

**本地 Whisper 配置：**
- 模型：medium（中文准确率与速度的平衡点）
- 设备：CPU（MVP 阶段）
- 并发：单服务器 2~4 个并发任务（视视频时长调整）
- 队列策略：付费用户优先，免费用户排队

### 4.5 AI 处理

| 技术 | 用途 | 成本估算 |
|------|------|----------|
| DeepSeek API | AI 总结、笔记生成 | 输入 ¥0.001/千字，输出 ¥0.002/千字 |
| Claude API（备用） | 复杂笔记场景降级 | 按需切换 |

**Prompt 策略：**
- 总结 Prompt：提取核心观点 + 章节摘要，控制输出长度在 1000 字内
- 笔记 Prompt：要求输出标准 Markdown，含标题层级、要点列表、代码块、补充说明框
- 问答 Prompt：RAG 模式，将转录文本切片作为 Context 传入

### 4.6 数据存储

|------|------|

### 4.7 基础设施

| 技术 | 用途 | 配置/成本 |

## 十、风险与应对

| 风险 | 影响 | 概率 | 应对方案 |
|------|------|------|----------|
| B 站字幕接口变动/反爬 | 高 | 中 | 优先使用官方开放接口；备用方案：引导用户手动上传视频文件 |
| Whisper CPU 处理速度慢，用户体验差 | 高 | 高 | 明确告知"预计等待时间"；付费用户优先队列；用户量达标后升级 GPU |
| 备案审核被驳回 | 中 | 中 | 提前准备完整材料；备案描述避免"爬虫/抓取"字眼，强调"视频笔记整理工具" |
| LLM API 涨价或不稳定 | 中 | 低 | DeepSeek 主力 + Claude 备用；Prompt 优化减少 Token 消耗 |
| 羊毛党/脚本滥用 | 中 | 中 | 设备指纹 + 行为检测 + 无限版公平使用上限 |
| 版权纠纷 | 中 | 低 | 用户协议明确仅限个人学习；不存储原视频；不提供下载功能 |
| 付费转化率低 | 高 | 中 | MVP 阶段先验证"用户是否愿意反复使用"，再优化转化漏斗 |
