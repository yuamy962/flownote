
# FlowNote 小红书配图制作流程文档

## 一、项目概述

### 1.1 需求背景
为小红书「创始人故事版」文案制作 9 张配图，用于推广 FlowNote AI 笔记工具。

### 1.2 目标用户
所有通过视频课程学习的人（AI、编程、设计、考研等）

### 1.3 设计风格
- **主色调**：深蓝色系，科技感
- **视觉风格**：简洁、专业、学习氛围
- **尺寸要求**：1080 × 1440 px（3:4 竖版）

## 四、详细制作流程

### 4.1 封面图制作（图1）

#### 步骤1：生成三屏底图
```
提示词（豆包生图）：
生成一张科技风格的三屏界面图，深色背景。左边是学习笔记界面（显示思维导图、彩色标签、知识点整理），中间是视频播放界面（显示课程内容），右边是笔记工具界面（显示结构化笔记）。三屏并列，带有圆角和投影效果。
```

#### 步骤2：用Canva排版
1. 创建 1080 × 1440 px 画布
2. 设置深蓝色渐变背景
3. 插入生成的三屏图，调整位置和大小
4. 添加白色主标题文字（72px，加粗）
5. 添加蓝色域名文字（36px）
6. 导出PNG格式

### 4.2 截图类图片制作（图3-图7）

#### 步骤1：准备截图素材
1. 打开浏览器，访问 flownote.cn
2. 登录测试账号
3. 准备一个B站视频链接用于演示

#### 步骤2：截取界面
1. 使用 Snipaste 截取各功能页面
2. 确保截图清晰，无遮挡
3. 截取时隐藏敏感信息

#### 步骤3：美化处理（可选）
1. 用 Canva 添加圆角和投影
2. 添加箭头标注重点功能
3. 添加简短文字说明

### 4.3 对比图制作（图6）

#### 步骤1：获取素材
- 原始字幕截图：从B站视频截取
- FlowNote笔记截图：从产品截取

#### 步骤2：用Canva拼接
1. 创建画布
2. 插入两张截图，左右排列
3. 添加分隔线和标签（"传统方式" vs "FlowNote"）
4. 添加对比说明文字

### 4.4 设计类图片制作（图2、图8、图9）

#### 步骤1：选择模板
- 在 Canva 搜索"小红书封面"或"对比图"模板
- 选择符合科技风格的模板

#### 步骤2：修改内容
1. 替换模板中的图片为产品相关内容
2. 修改文字为宣传文案
3. 调整配色为深蓝系

## 五、自动化方案（进阶）

### 5.1 AI批量生成脚本

```python
# Python 脚本示例（需要安装相关库）
import requests

def generate_image(prompt):
    """调用豆包生图API生成图片"""
    url = "https://api.doubao.com/text2image"
    payload = {
        "prompt": prompt,
        "size": "1080x1440",
        "style": "科技风"
    }
    response = requests.post(url, json=payload)
    return response.json()["image_url"]

# 批量生成配图
prompts = [
    "科技风格三屏界面，学习笔记+视频+AI笔记",
    "学习场景痛点，视频暂停记笔记",
    "AI工具定价方案对比图"
]

for i, prompt in enumerate(prompts):
    image_url = generate_image(prompt)
    print(f"图{i+1}: {image_url}")
```

### 5.2 模板替换自动化

使用 Figma API 实现：
1. 在 Figma 中创建配图模板
2. 通过 API 动态替换文字和图片
3. 自动导出为 PNG

## 六、质量检查清单

### 6.1 图片规范检查

| 检查项 | 要求 |
|--------|------|
| 尺寸 | 1080 × 1440 px |
| 格式 | PNG（透明背景）或 JPG |
| 文件大小 | < 5MB |
| 分辨率 | 72 dpi |

### 6.2 内容规范检查

| 检查项 | 要求 |
|--------|------|
| 文字清晰 | 主要文字 > 24px |
| 配色统一 | 深蓝色系为主 |
| 信息准确 | 价格、功能描述无误 |
| 品牌露出 | 每张图包含 flownote.cn |

### 6.3 合规性检查

| 检查项 | 要求 |
|--------|------|
| 无竞品名称 | 不提及其他产品 |
| 无极限词 | 避免"最"、"第一"等 |
| 无侵权内容 | 截图使用自有产品 |

## 八、常见问题

### Q1：AI生成的图片不符合预期怎么办？
- **解决方案**：调整提示词，增加细节描述；或手动用设计工具修改。

### Q2：截图中的敏感信息如何处理？
- **解决方案**：使用测试账号截图；用图片编辑工具模糊敏感区域。

### Q3：如何保持9张图风格统一？
- **解决方案**：使用同一套配色和字体；创建统一模板。
